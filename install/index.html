Server requirements (important!)

What do I need to run this system?

Folder Permissions

• /public/img/ needs to be writable (see chmod)
• /public/themes/ is writable

PHP Version

Minimum required PHP version: 5.3

PHP Extensions

The following PHP extensions are required:
php_mysqli, pdo_mysql, php_gd, php_json
The following modules are missing:
How do I enable PHP extensions?
Go into your PHP directory and find the php.ini file. Mine was located in C:\wamp\bin\php\php5.3.13 OR C:\AppServ\php5. Open the file with a text editor and search (CTRL+F) for one of the modules you need to enable. To enable them, simply remove the ; character in front of the line.
Save the file and restart your webserver to apply the changes.

Apache Modules

We recommend ConquerCMS in combination with an Apache Webserver.

The Apache modules you'll need are:
mod_rewrite, mod_gzip
The following modules are missing:
How do I enable Apache modules?
Go into your Apache directory and find the httpd.conf file. Mine was located in Apache\conf\. Open the file with a text editor and search CTRL+F for one of the modules you need to enable. To enable them, simply remove the # character in front of the line.
Save the file and restart your webserver to apply the changes.
